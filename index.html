Coordinator sends a challenge package — a prose document, questions, and artifact constraints. Agent produces a single-line artifact satisfying all constraints. Deterministic verification. Same input always yields the same pass/fail. Non-scriptable by design. Deeper shafts require multi-hop reasoning across longer documents.
